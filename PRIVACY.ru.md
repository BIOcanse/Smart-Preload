Эта политика применяется к браузерному расширению Smart Preload и необязательному сопутствующему приложению для Windows.

Smart Preload использует интеллектуальные алгоритмы предварительной загрузки, чтобы уменьшить воспринимаемое время ожидания. Для этого расширение обрабатывает связанные с просмотром сигналы в профиле браузера и на вашем устройстве. Разработчик не использует сервер для сбора истории посещений, не продает пользовательские данные и не передает пользовательские данные рекламодателям, аналитическим брокерам или брокерам данных.

## Данные, обрабатываемые локально

Smart Preload может локально обрабатывать и хранить следующие данные:

- URL страниц, хосты, заголовки и переходы навигации.
- Найденные на страницах ссылки-кандидаты, включая URL ссылок, текст ссылок и близлежащий текст, используемый для ранжирования.
- Состояние вкладок, окон, предварительной загрузки и prefetch, необходимое для управления подготовленными страницами.
- Сигналы взаимодействия, такие как наведение на ссылку, намерение предварительной загрузки через контекстное меню, активность вкладки на переднем плане, недавнее активное время и состояние медиа/активности, используемое планировщиком.
- Названия и URL закладок, если включены функции предварительной загрузки на основе закладок.
- Настройки расширения, лимиты предварительной загрузки, настройки безопасности, локальная статистика истории, настройки AI-провайдера и диагностические журналы, если диагностика включена.

Эти локальные данные используются только для функций Smart Preload: прогнозирования, ранжирования, планирования предварительной загрузки, фильтрации безопасности, удаления локальной истории, диагностики и необязательной AI-оценки.

## Данные, отправляемые за пределы устройства

Smart Preload не отправляет историю посещений или историю предварительной загрузки разработчику.

Данные могут покинуть ваше устройство только в следующих случаях:

- Если вы включаете внешнего AI-провайдера и вводите API key или endpoint, расширение может отправить этому провайдеру выбранный контекст страницы/ссылки, необходимый для оценки ключевых слов или релевантности. К таким запросам применяется политика конфиденциальности соответствующего провайдера.
- Если вы используете локальный AI endpoint, например LM Studio, запросы отправляются на настроенный вами endpoint.
- Если вы проверяете или загружаете обновления нативного приложения, расширение или сопутствующее приложение Windows может обращаться к страницам GitHub releases или файлам, размещенным на GitHub.
- Когда вы действительно посещаете страницу или когда функция предварительной загрузки загружает страницу в браузере, происходит обычная сетевая активность браузера. Целевой сайт может получать обычные запросы, cookies и информацию о сессии, как при обычной загрузке страницы.

## Необязательное сопутствующее приложение Windows

Необязательное сопутствующее приложение Windows помогает держать реальные окна предварительной загрузки скрытыми и поддерживает локальную системную интеграцию. Оно взаимодействует с расширением на том же устройстве через Chrome/Edge native messaging. Оно может локально обрабатывать метаданные окон браузера и сведения о производительности/активности системы для управления предварительной загрузкой. Оно не отправляет историю посещений разработчику.

## Хранение и удаление

Smart Preload хранит данные в локальном хранилище расширения браузера и связанных локальных файлах, используемых расширением или сопутствующим приложением. Вы можете удалить выбранные диапазоны локальной истории на странице настроек расширения. Вы также можете удалить сохраненные данные, удалив расширение или очистив данные расширения в профиле браузера.

## Разрешения

Smart Preload запрашивает разрешения браузера только для поддержки своих функций, включая обработку вкладок и событий навигации, локальное хранилище, функции предварительной загрузки на основе закладок, native messaging с необязательным сопутствующим приложением и плановые проверки обслуживания.

## Без продажи и рекламного использования

Разработчик не продает, не сдает в аренду и не передает пользовательские данные рекламодателям, аналитическим брокерам или другим третьим сторонам для отслеживания, профилирования или рекламы.

## Limited Use Chrome Web Store

Использование Smart Preload информации, полученной через API расширений Chrome, ограничено предоставлением и улучшением единственной цели: уменьшать воспринимаемое время ожидания загрузки страниц с помощью локального прогнозирования и предварительной загрузки. Использование этой информации в Smart Preload соответствует Chrome Web Store User Data Policy, включая требования Limited Use.

## Конфиденциальность детей

Smart Preload не предназначен для детей и сознательно не собирает личную информацию детей.

## Изменения

Эта политика может обновляться при изменении обработки данных в Smart Preload. Последняя версия будет опубликована в этом репозитории.
