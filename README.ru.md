# Smart Preload / Zero Latency Web

[English](README.md) | [简体中文](README.zh-CN.md) | [繁體中文](README.zh-TW.md) | [日本語](README.ja.md) | [한국어](README.ko.md) | [Deutsch](README.de.md) | [Français](README.fr.md) | [Español](README.es.md) | [Português (Brasil)](README.pt-BR.md) | Русский

Smart Preload - расширение для Chrome и Edge, которое активнее предварительно загружает страницы, которые вы, скорее всего, откроете следующими. Оно объединяет оценку ссылок, историю активности вкладок, предзагрузку при наведении и Windows-приложение для более надежного управления фоновыми окнами.

## Загрузка

Скачайте последнюю версию на странице [GitHub Releases](https://github.com/kingstonwang114514-cloud/zero-latency-web/releases/latest).

Windows-приложение необязательно, но рекомендуется. Оно доступно только для Windows и используется для Native Messaging, восстановления watchdog, снимков производительности и управления фоновыми окнами.

## Установка

1. Сначала установите или включите расширение в Chrome или Edge.
2. Распакуйте пакет Windows-приложения.
3. Запустите `install-register.cmd` из распакованной папки app или один раз запустите приложение.
4. После первой успешной привязки расширение сможет автоматически запускать локальное приложение.

Если приложение не обнаруживается около одной минуты, расширение может предложить скачать приложение или включить полностью нативный режим предзагрузки.

## Возможности

- Глобальное планирование предзагрузки по видимым вкладкам, а не только по текущей вкладке.
- Раздельные лимиты для обычной предзагрузки и настоящей предзагрузки фоновых вкладок.
- Независимая предзагрузка при наведении на ссылку или при открытии контекстного меню ссылки.
- Исключения для локальных страниц, страниц частной сети, страниц Google, инкогнито-окон и настроенного прокси-браузинга.
- Ручной выбор языка интерфейса в дополнение к автоматическому определению языка браузера.
- Локальные исторические данные хранятся в переносимой папке и могут быть скопированы на новый компьютер или в новую версию.

## Примечания

- Поддерживаемые браузеры: Chromium-версии Google Chrome и Microsoft Edge.
- Нативное приложение: только Windows.
- Порядок первой привязки важен: сначала установите или включите расширение, затем выполните регистрацию app.
- Логика прогнозирования остается в расширении. Локальное приложение не запускает локальные AI-модели и не хранит ключи AI-провайдеров.
