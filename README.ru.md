# Smart Preload / Zero Latency Web

[English](README.md) | [简体中文](README.zh-CN.md) | [繁體中文](README.zh-TW.md) | [日本語](README.ja.md) | [한국어](README.ko.md) | [Deutsch](README.de.md) | [Français](README.fr.md) | [Español](README.es.md) | [Português (Brasil)](README.pt-BR.md) | Русский

Smart Preload заранее готовит страницы, которые вы, скорее всего, откроете следующими, чтобы переходы ощущались быстрее. Он подходит для поиска информации, работы, чтения с множеством вкладок, сравнения покупок и просмотра документации.

Smart Preload незаметно работает во время просмотра, заранее подготавливает полезные страницы и оставляет контроль над приватностью, производительностью и поведением браузера.

## Что вы получите

- Более быстрые переходы, когда следующая страница уже подготовлена.
- Более удобную работу с большим количеством вкладок, а не только с текущей страницей.
- Подготовку ссылок при наведении мыши или открытии контекстного меню.
- Раздельное управление обычной предзагрузкой и более активной фоновой предзагрузкой.
- Возможность исключать локальные страницы, страницы частной сети, страницы Google, окна инкогнито и настроенный прокси-браузинг.
- Автоматический выбор языка интерфейса или ручную настройку языка.
- Исторические данные, которые можно перенести папкой на новый компьютер или в новую версию.

## Загрузка

Скачайте последнюю версию на странице [GitHub Releases](https://github.com/kingstonwang114514-cloud/zero-latency-web/releases/latest).

Smart Preload может работать только как расширение браузера. Необязательное Windows-приложение улучшает локальную интеграцию и помогает расширению и приложению автоматически восстанавливать связь при необходимости. Приложение доступно только для Windows.

## Первая настройка

1. Сначала установите или включите расширение в Chrome или Edge.
2. Если хотите использовать Windows-приложение, распакуйте пакет приложения.
3. Запустите `install-register.cmd` из распакованной папки app или один раз запустите приложение.
4. После первой успешной привязки следующие запуски смогут переподключаться автоматически.

Если приложение не обнаруживается около одной минуты, расширение может предложить скачать приложение или использовать режим предзагрузки только средствами браузера.

## Поддержка браузеров

- Google Chrome
- Microsoft Edge
- Другие браузеры на базе Chromium могут работать, но основными целями являются Chrome и Edge.
