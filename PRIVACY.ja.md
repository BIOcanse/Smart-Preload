# Smart Preload プライバシーポリシー

最終更新日: 2026 年 6 月 23 日

このポリシーは、Smart Preload ブラウザー拡張機能および任意で利用できる Windows 連携アプリに適用されます。

Smart Preload は、体感上の読み込み待ち時間を減らすためにインテリジェントなプリロードアルゴリズムを使用します。そのため、ブラウザープロファイル内および端末上で、閲覧に関連する信号を処理します。開発者は閲覧履歴を収集するサーバーを運用しておらず、ユーザーデータを販売せず、広告事業者、分析データブローカー、データブローカーと共有しません。

## ローカルで処理されるデータ

Smart Preload は、次のデータをローカルで処理および保存する場合があります。

- ページ URL、ホスト、タイトル、ナビゲーション遷移。
- ページ上で見つかったリンク候補、リンク URL、アンカーテキスト、ランキングに使う周辺テキスト。
- 準備済みページを管理するためのタブ、ウィンドウ、プリロード、プリフェッチ状態。
- リンクのホバー、コンテキストメニューのプリロード意図、前面タブの活動、最近のアクティブ時間、スケジューリングに使うメディア/活動状態。
- ブックマークベースのプリロード機能を有効にした場合のブックマークのタイトルと URL。
- 拡張機能の設定、プリロード上限、安全設定、ローカル履歴統計、AI プロバイダー設定、診断を有効にした場合の診断ログ。

これらのローカルデータは、予測、ランキング、プリロードスケジューリング、安全フィルタリング、ローカル履歴削除、診断、任意の AI 支援スコアリングのためにのみ使用されます。

## 端末外に送信されるデータ

Smart Preload は、閲覧履歴やプリロード履歴を開発者に送信しません。

データが端末外に出る可能性があるのは次の場合だけです。

- 外部 AI プロバイダーを有効にし、API key またはエンドポイントを入力した場合、拡張機能はキーワードまたは関連性スコアリングに必要なページ/リンクのコンテキストをそのプロバイダーに送信する場合があります。そのリクエストには、そのプロバイダーのプライバシーポリシーが適用されます。
- LM Studio などのローカル AI エンドポイントを使用する場合、リクエストは設定したエンドポイントに送信されます。
- Windows 連携アプリの更新を確認またはダウンロードする場合、拡張機能または Windows 連携アプリが GitHub のリリースページまたは GitHub がホストするファイルにアクセスする場合があります。
- 実際にページを訪問する場合、またはプリロード機能がブラウザー内でページを読み込む場合、通常のブラウザーネットワーク通信が発生します。対象サイトは通常のページ読み込みと同様に、リクエスト、Cookie、セッション情報を受け取る場合があります。

## 任意の Windows 連携アプリ

任意の Windows 連携アプリは、実際のプリロードウィンドウを隠したまま保ち、ローカルシステム連携を支援します。同じ端末上で Chrome/Edge native messaging を通じて拡張機能と通信します。プリロード動作を管理するために、ローカルのブラウザーウィンドウメタデータやシステム性能/活動情報を処理する場合があります。閲覧履歴を開発者に送信することはありません。

## 保存と削除

Smart Preload は、ブラウザー拡張機能のローカルストレージ、および拡張機能または連携アプリが使う関連ローカルファイルにデータを保存します。拡張機能の設定ページから、指定した期間のローカル履歴を削除できます。また、拡張機能をアンインストールするか、ブラウザープロファイルの拡張機能データを消去することで、保存済みデータを削除できます。

## 権限

Smart Preload は、タブとナビゲーションイベントの処理、ローカルストレージ、ブックマークベースのプリロード、任意の連携アプリとの native messaging、定期メンテナンスチェックなど、機能提供に必要な権限のみを要求します。

## 販売や広告利用はしません

開発者は、追跡、プロファイリング、広告のために、ユーザーデータを販売、貸与、または広告事業者、分析データブローカー、その他の第三者と共有しません。

## Chrome ウェブストアの Limited Use

Smart Preload が Chrome 拡張 API から受け取る情報の使用は、ローカル予測とプリロードによって体感上のページ読み込み待ち時間を減らすという単一目的の提供および改善に限定されます。Smart Preload によるこれらの情報の使用は、Limited Use 要件を含む Chrome Web Store User Data Policy に従います。

## 児童のプライバシー

Smart Preload は児童を対象としておらず、児童の個人情報を故意に収集しません。

## 変更

Smart Preload のデータ取り扱いが変わる場合、このポリシーを更新することがあります。最新版はこのリポジトリで公開されます。
