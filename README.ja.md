# Smart Preload / Zero Latency Web

[English](README.md) | [简体中文](README.zh-CN.md) | [繁體中文](README.zh-TW.md) | 日本語 | [한국어](README.ko.md) | [Deutsch](README.de.md) | [Français](README.fr.md) | [Español](README.es.md) | [Português (Brasil)](README.pt-BR.md) | [Русский](README.ru.md)

Smart Preload は、次に開く可能性が高いページを先に準備し、ページ移動をより速く感じられるようにする拡張機能です。調査、仕事、複数タブでの読書、買い物の比較、ドキュメント閲覧など、ページを頻繁に行き来する使い方に向いています。

ブラウジングを邪魔せず、必要になりそうなページを自然に先回りして準備します。同時に、プライバシー、性能、ブラウジング動作の調整もできます。

## 得られる体験

- 次のページが準備済みなら、移動がより速く自然に感じられます。
- 現在のページだけでなく、複数タブを使う作業にも合わせて動きます。
- リンクにマウスを置いたり右クリックしたりした時点で、開く可能性のあるページを準備できます。
- 通常のプリロードと、より積極的なバックグラウンドプリロードを別々に調整できます。
- ローカルページ、プライベートネットワークページ、Google ページ、シークレットウィンドウ、設定したプロキシ関連の閲覧を除外できます。
- UI 言語は自動判定と手動選択の両方に対応します。
- 履歴データはフォルダーごと移行でき、新しい PC や新しいバージョンでも使い方を引き継ぎやすくなります。

## ダウンロード

最新版は [GitHub Releases](https://github.com/kingstonwang114514-cloud/zero-latency-web/releases/latest) からダウンロードしてください。

Smart Preload はブラウザ拡張機能だけでも利用できます。任意の Windows アプリを使うと、ローカル環境との連携が強化され、必要なときに拡張機能とアプリが自動で再接続しやすくなります。アプリは Windows のみ対応です。

## 初回設定

1. 先に Chrome または Edge で拡張機能をインストールまたは有効化します。
2. Windows 連携アプリを使う場合は、アプリパッケージを展開します。
3. 展開した app フォルダーで `install-register.cmd` を実行するか、アプリを一度起動します。
4. 初回の連携が成功すると、以後は自動で再接続できます。

約 1 分間アプリを検出できない場合、拡張機能はアプリのダウンロードまたはブラウザのみのプリロードモードを案内できます。

## 対応ブラウザ

- Google Chrome
- Microsoft Edge
- その他の Chromium ベースブラウザでも動作する可能性がありますが、主な対象は Chrome と Edge です。
