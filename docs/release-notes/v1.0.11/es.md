# Smart Preload v1.0.11

[English](en.md) | [简体中文](zh-CN.md) | [繁體中文](zh-TW.md) | [日本語](ja.md) | [한국어](ko.md) | [Deutsch](de.md) | [Français](fr.md) | [Español](es.md) | [Português (Brasil)](pt-BR.md) | [Русский](ru.md)

Smart Preload es una extension de navegador para precargar paginas de forma mas inteligente y proactiva.

Descargas:

- Paquete completo: `zero-latency-web-release-v1.0.11.zip`
- Solo extension: `zero-latency-web-extension-v1.0.11.zip`
- Solo app nativa: `zero-latency-web-app-windows-x64-v1.0.11.zip`

App nativa: solo Windows.

Primer enlace: instala o habilita primero la extension y despues ejecuta el instalador de la app de Windows o inicia la app de Windows. Despues del enlace, la extension puede iniciar la app nativa automaticamente.

Nuevo en esta version:

- Se agrego seleccion manual del idioma de la interfaz.
- Las notas de GitHub release ahora estan separadas por idioma.
- Los assets publicos de GitHub release se limitan al paquete completo, extension sola y app de Windows sola.
