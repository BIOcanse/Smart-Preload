# Smart Preload v1.0.11

[English](en.md) | [简体中文](zh-CN.md) | [繁體中文](zh-TW.md) | [日本語](ja.md) | [한국어](ko.md) | [Deutsch](de.md) | [Français](fr.md) | [Español](es.md) | [Português (Brasil)](pt-BR.md) | [Русский](ru.md)

Smart Preload は、次に開く可能性が高いページをより賢く積極的にプリロードするブラウザ拡張機能です。

ダウンロード:

- フルパッケージ: `zero-latency-web-release-v1.0.11.zip`
- 拡張機能のみ: `zero-latency-web-extension-v1.0.11.zip`
- ネイティブアプリのみ: `zero-latency-web-app-windows-x64-v1.0.11.zip`

ネイティブアプリ対応: Windows のみ。

初回連携手順: 先に拡張機能をインストールまたは有効化し、その後 Windows アプリのインストーラーを実行するか Windows アプリを起動してください。連携後は拡張機能がネイティブアプリを自動起動できます。

このバージョンの変更:

- UI 言語を手動選択できるようになりました。
- GitHub リリースノートを言語別ページに分割しました。
- 公開 GitHub release のアセットは、フルパッケージ、拡張機能のみ、Windows アプリのみの 3 種類です。
