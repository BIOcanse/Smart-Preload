# Smart Preload v1.0.11

[English](en.md) | [简体中文](zh-CN.md) | [繁體中文](zh-TW.md) | [日本語](ja.md) | [한국어](ko.md) | [Deutsch](de.md) | [Français](fr.md) | [Español](es.md) | [Português (Brasil)](pt-BR.md) | [Русский](ru.md)

Smart Preload 是用於更智慧、更積極預載網頁的瀏覽器擴充功能。

下載：

- 組合包：`zero-latency-web-release-v1.0.11.zip`
- 單擴充功能包：`zero-latency-web-extension-v1.0.11.zip`
- 單本機 app 包：`zero-latency-web-app-windows-x64-v1.0.11.zip`

本機 app 支援範圍：僅 Windows。

首次綁定順序：先安裝或啟用擴充功能，再執行 Windows app 安裝腳本或啟動 Windows app。綁定成功後，擴充功能可以自動喚起本機 app。
