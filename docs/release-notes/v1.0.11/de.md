# Smart Preload v1.0.11

[English](en.md) | [简体中文](zh-CN.md) | [繁體中文](zh-TW.md) | [日本語](ja.md) | [한국어](ko.md) | [Deutsch](de.md) | [Français](fr.md) | [Español](es.md) | [Português (Brasil)](pt-BR.md) | [Русский](ru.md)

Smart Preload ist eine Browsererweiterung fuer intelligenteres und aktiveres Vorladen von Seiten.

Downloads:

- Komplettpaket: `zero-latency-web-release-v1.0.11.zip`
- Nur Erweiterung: `zero-latency-web-extension-v1.0.11.zip`
- Nur native App: `zero-latency-web-app-windows-x64-v1.0.11.zip`

Native App: nur Windows.

Erste Kopplung: Installieren oder aktivieren Sie zuerst die Erweiterung, starten Sie danach das Windows-App-Installationsskript oder die Windows-App. Nach erfolgreicher Kopplung kann die Erweiterung die native App automatisch starten.

Neu in dieser Version:

- Manuelle Auswahl der UI-Sprache wurde hinzugefuegt.
- GitHub Release Notes sind jetzt in getrennte Sprachseiten aufgeteilt.
- Oeffentliche GitHub Release Assets sind auf Komplettpaket, Erweiterung und Windows App begrenzt.
