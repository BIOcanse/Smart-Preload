# Smart Preload v1.0.11

[English](en.md) | [简体中文](zh-CN.md) | [繁體中文](zh-TW.md) | [日本語](ja.md) | [한국어](ko.md) | [Deutsch](de.md) | [Français](fr.md) | [Español](es.md) | [Português (Brasil)](pt-BR.md) | [Русский](ru.md)

Smart Preload est une extension de navigateur pour un prechargement plus intelligent et plus actif des pages.

Telechargements :

- Package complet : `zero-latency-web-release-v1.0.11.zip`
- Extension seule : `zero-latency-web-extension-v1.0.11.zip`
- Application native seule : `zero-latency-web-app-windows-x64-v1.0.11.zip`

Application native : Windows uniquement.

Premier lien : installez ou activez d'abord l'extension, puis lancez le script d'installation de l'application Windows ou demarrez l'application Windows. Une fois le lien etabli, l'extension peut demarrer automatiquement l'application native.

Nouveautes :

- Ajout de la selection manuelle de la langue de l'interface.
- Les notes de version GitHub sont maintenant separees par langue.
- Les assets publics GitHub release sont limites au package complet, a l'extension seule et a l'application Windows seule.
