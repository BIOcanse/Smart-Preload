# Smart Preload v1.0.11

[English](en.md) | [简体中文](zh-CN.md) | [繁體中文](zh-TW.md) | [日本語](ja.md) | [한국어](ko.md) | [Deutsch](de.md) | [Français](fr.md) | [Español](es.md) | [Português (Brasil)](pt-BR.md) | [Русский](ru.md)

Smart Preload는 다음에 열 가능성이 높은 페이지를 더 스마트하고 적극적으로 미리 불러오는 브라우저 확장 프로그램입니다.

다운로드:

- 전체 패키지: `zero-latency-web-release-v1.0.11.zip`
- 확장 프로그램만: `zero-latency-web-extension-v1.0.11.zip`
- 네이티브 앱만: `zero-latency-web-app-windows-x64-v1.0.11.zip`

네이티브 앱 지원: Windows 전용.

첫 연결 순서: 확장 프로그램을 먼저 설치하거나 활성화한 뒤 Windows 앱 설치 스크립트를 실행하거나 Windows 앱을 시작하세요. 연결 후에는 확장 프로그램이 네이티브 앱을 자동으로 실행할 수 있습니다.

이번 버전 변경:

- UI 언어를 수동으로 선택할 수 있습니다.
- GitHub 릴리스 노트를 언어별 페이지로 분리했습니다.
- 공개 GitHub release 자산은 전체 패키지, 확장 프로그램 전용, Windows 앱 전용 세 가지입니다.
