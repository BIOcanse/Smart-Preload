# Smart Preload v1.0.11

[English](en.md) | [简体中文](zh-CN.md) | [繁體中文](zh-TW.md) | [日本語](ja.md) | [한국어](ko.md) | [Deutsch](de.md) | [Français](fr.md) | [Español](es.md) | [Português (Brasil)](pt-BR.md) | [Русский](ru.md)

Smart Preload - расширение браузера для более умной и активной предварительной загрузки страниц.

Загрузки:

- Полный пакет: `zero-latency-web-release-v1.0.11.zip`
- Только расширение: `zero-latency-web-extension-v1.0.11.zip`
- Только нативное приложение: `zero-latency-web-app-windows-x64-v1.0.11.zip`

Нативное приложение: только Windows.

Первое связывание: сначала установите или включите расширение, затем запустите установочный скрипт Windows-приложения или само Windows-приложение. После успешного связывания расширение сможет автоматически запускать нативное приложение.

Новое в версии:

- Добавлен ручной выбор языка интерфейса.
- Примечания к GitHub release разделены на отдельные языковые страницы.
- Публичные assets GitHub release ограничены полным пакетом, расширением и Windows-приложением.
