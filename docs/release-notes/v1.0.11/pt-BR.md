# Smart Preload v1.0.11

[English](en.md) | [简体中文](zh-CN.md) | [繁體中文](zh-TW.md) | [日本語](ja.md) | [한국어](ko.md) | [Deutsch](de.md) | [Français](fr.md) | [Español](es.md) | [Português (Brasil)](pt-BR.md) | [Русский](ru.md)

Smart Preload e uma extensao de navegador para pre-carregar paginas de forma mais inteligente e proativa.

Downloads:

- Pacote completo: `zero-latency-web-release-v1.0.11.zip`
- Somente extensao: `zero-latency-web-extension-v1.0.11.zip`
- Somente app nativo: `zero-latency-web-app-windows-x64-v1.0.11.zip`

App nativo: somente Windows.

Primeira vinculacao: instale ou ative primeiro a extensao e depois execute o instalador do app Windows ou inicie o app Windows. Depois da vinculacao, a extensao pode iniciar o app nativo automaticamente.

Novidades:

- Adicionada selecao manual do idioma da interface.
- As notas do GitHub release agora ficam em paginas separadas por idioma.
- Os assets publicos do GitHub release se limitam ao pacote completo, extensao somente e app Windows somente.
