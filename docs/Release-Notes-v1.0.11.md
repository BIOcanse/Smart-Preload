<details name="release-language" open>
<summary><strong>English</strong></summary>

Smart Preload is a browser extension for smarter, more proactive page preloading.

Downloads:

- Full package: `zero-latency-web-release-v1.0.11.zip`
- Extension only: `zero-latency-web-extension-v1.0.11.zip`
- Native app only: `zero-latency-web-app-windows-x64-v1.0.11.zip`

Native app support: Windows only.

First binding order: install or enable the extension first, then run the Windows app installer or start the Windows app. After binding succeeds, the extension can wake the native app automatically.

New in this version:

- Added manual UI language selection: automatic, English, Simplified Chinese, Traditional Chinese, Japanese, Korean, German, French, Spanish, Portuguese (Brazil), and Russian.
