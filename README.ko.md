# Smart Preload / Zero Latency Web

[English](README.md) | [简体中文](README.zh-CN.md) | [繁體中文](README.zh-TW.md) | [日本語](README.ja.md) | 한국어 | [Deutsch](README.de.md) | [Français](README.fr.md) | [Español](README.es.md) | [Português (Brasil)](README.pt-BR.md) | [Русский](README.ru.md)

Smart Preload는 다음에 열 가능성이 높은 페이지를 미리 준비해 페이지 이동이 더 빠르게 느껴지도록 하는 확장 프로그램입니다. 자료 조사, 업무, 여러 탭 읽기, 쇼핑 비교, 문서 확인처럼 페이지를 자주 오가는 상황에 적합합니다.

브라우징을 방해하지 않으면서 곧 필요할 가능성이 높은 페이지를 자연스럽게 먼저 준비합니다. 동시에 개인 정보, 성능, 브라우징 동작에 대한 제어도 제공합니다.

## 사용자가 얻는 경험

- 다음 페이지가 이미 준비되어 있으면 이동이 더 빠르고 자연스럽게 느껴집니다.
- 현재 페이지만이 아니라 여러 탭을 오가는 작업 흐름에도 맞춰 동작합니다.
- 링크에 마우스를 올리거나 우클릭하는 순간, 열 가능성이 높은 페이지를 미리 준비할 수 있습니다.
- 일반 프리로드와 더 적극적인 백그라운드 프리로드를 따로 조절할 수 있습니다.
- 로컬 페이지, 사설 네트워크 페이지, Google 페이지, 시크릿 창, 설정한 프록시 관련 탐색을 제외할 수 있습니다.
- UI 언어는 자동 감지와 수동 선택을 모두 지원합니다.
- 기록 데이터는 폴더째 옮길 수 있어 새 컴퓨터나 새 버전에서도 사용 습관을 이어가기 쉽습니다.

## 다운로드

최신 버전은 [GitHub Releases](https://github.com/kingstonwang114514-cloud/zero-latency-web/releases/latest)에서 다운로드하세요.

Smart Preload는 브라우저 확장 프로그램만으로도 사용할 수 있습니다. 선택 사항인 Windows 앱을 사용하면 로컬 연동이 더 좋아지고, 필요할 때 확장 프로그램과 앱이 자동으로 다시 연결되기 쉬워집니다. 앱은 Windows 전용입니다.

## 첫 설정

1. 먼저 Chrome 또는 Edge에서 확장 프로그램을 설치하거나 활성화합니다.
2. Windows 연동 앱을 사용하려면 앱 패키지를 압축 해제합니다.
3. 압축을 푼 app 폴더에서 `install-register.cmd`를 실행하거나 앱을 한 번 시작합니다.
4. 첫 연결이 성공하면 이후에는 자동으로 다시 연결할 수 있습니다.

약 1분 동안 앱을 감지하지 못하면 확장 프로그램이 앱 다운로드 또는 브라우저 전용 프리로드 모드를 안내할 수 있습니다.

## 브라우저 지원

- Google Chrome
- Microsoft Edge
- 기타 Chromium 기반 브라우저에서도 동작할 수 있지만, 주요 대상은 Chrome과 Edge입니다.
