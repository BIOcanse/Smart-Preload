# 智慧預載隱私權政策

最後更新：2026 年 6 月 23 日

本政策適用於智慧預載瀏覽器擴充功能，以及可選的 Windows 本機配套 app。

智慧預載使用智慧預載演算法來降低使用者感知到的載入等待時間。為此，它會在你的瀏覽器設定檔和本機裝置上處理與瀏覽相關的訊號。開發者沒有用來收集瀏覽歷史的伺服器，不會出售使用者資料，也不會與廣告商、分析資料經紀商或資料經紀商共享使用者資料。

## 本機處理的資料

智慧預載可能會在本機處理並儲存以下資料：

- 頁面 URL、站點、標題和導覽跳轉關係。
- 頁面中的連結候選，包括連結 URL、連結文字，以及用於排序的附近文字。
- 管理已準備頁面所需的分頁、視窗、預載和預取狀態。
- 連結懸停、右鍵選單預載意圖、前景分頁活動、近期活躍時間，以及調度所需的媒體/活動狀態。
- 啟用書籤預載功能時的書籤標題和 URL。
- 擴充功能設定、預載上限、安全設定、本機歷史統計、AI 服務設定，以及啟用診斷時的診斷記錄。

這些本機資料只用於提供智慧預載功能：預測、排序、預載調度、安全過濾、本機歷史刪除、診斷和可選的 AI 輔助評分。

## 會離開裝置的資料

智慧預載不會把瀏覽歷史或預載歷史傳送給開發者。

只有以下情況資料可能離開你的裝置：

- 如果你啟用外部 AI 服務並填入 API key 或端點，擴充功能可能會把關鍵字或相關性評分所需的頁面/連結上下文傳送給該服務。該服務自己的隱私權政策適用於這些請求。
- 如果你使用 LM Studio 等本機 AI 端點，請求會傳送到你設定的端點。
- 如果你檢查或下載本機 app 更新，擴充功能或 Windows 本機配套 app 可能會存取 GitHub release 頁面或 GitHub 託管檔案。
- 當你實際造訪頁面，或預載功能在瀏覽器中載入頁面時，瀏覽器會進行正常網路請求。目標網站可能會像一般頁面載入一樣接收請求、Cookie 和工作階段資訊。

## 可選 Windows 本機 app

可選的 Windows 本機配套 app 用於保持真實預載視窗隱藏，並支援本機系統整合。它透過 Chrome/Edge native messaging 在同一台裝置上與擴充功能通訊。它可能會在本機處理瀏覽器視窗中繼資料和本機系統效能/活動資訊，以管理預載行為。它不會把瀏覽歷史傳送給開發者。

## 儲存和刪除

智慧預載的資料儲存在瀏覽器擴充功能本機儲存，以及擴充功能或本機 app 使用的相關本機檔案中。你可以在擴充功能設定頁刪除指定時間範圍內的本機歷史記錄。你也可以透過解除安裝擴充功能或清理瀏覽器設定檔中的擴充功能資料來移除已儲存資料。

## 權限

智慧預載只為支援功能而請求瀏覽器權限，包括分頁和導覽事件處理、本機儲存、基於書籤的預載功能、與可選本機 app 的 native messaging 通訊，以及定期維護檢查。

## 不出售或用於廣告

開發者不會把使用者資料出售、出租或共享給廣告商、分析資料經紀商或其他第三方用於追蹤、側寫或廣告。

## Chrome 線上應用程式商店 Limited Use

智慧預載使用從 Chrome 擴充功能 API 取得的資訊時，僅限於提供和改進其單一目的：透過本機預測和預載降低使用者感知到的頁面載入等待時間。智慧預載對這些資訊的使用遵守 Chrome Web Store User Data Policy，包括其中的 Limited Use 要求。

## 兒童隱私

智慧預載並非面向兒童，也不會有意收集兒童個人資訊。

## 政策更新

當智慧預載的資料處理方式發生變化時，本政策可能會更新。最新版本會發布在本倉庫中。
