# Smart Preload 개인정보 처리방침

마지막 업데이트: 2026년 6월 23일

[English](PRIVACY.md) | [简体中文](PRIVACY.zh-CN.md) | [繁體中文](PRIVACY.zh-TW.md) | [日本語](PRIVACY.ja.md) | 한국어 | [Deutsch](PRIVACY.de.md) | [Français](PRIVACY.fr.md) | [Español](PRIVACY.es.md) | [Português (Brasil)](PRIVACY.pt-BR.md) | [Русский](PRIVACY.ru.md)

이 정책은 Smart Preload 브라우저 확장 프로그램과 선택 사항인 Windows 동반 앱에 적용됩니다.

Smart Preload는 체감 로딩 대기 시간을 줄이기 위해 지능형 프리로드 알고리즘을 사용합니다. 이를 위해 브라우저 프로필과 사용자의 기기에서 브라우징 관련 신호를 처리합니다. 개발자는 브라우징 기록을 수집하는 서버를 운영하지 않으며, 사용자 데이터를 판매하지 않고, 광고주, 분석 데이터 브로커 또는 데이터 브로커와 공유하지 않습니다.

## 로컬에서 처리되는 데이터

Smart Preload는 다음 데이터를 로컬에서 처리하고 저장할 수 있습니다.

- 페이지 URL, 호스트, 제목, 탐색 전환.
- 페이지에서 발견된 링크 후보, 링크 URL, 앵커 텍스트, 순위 계산에 사용되는 주변 텍스트.
- 준비된 페이지를 관리하는 데 필요한 탭, 창, 프리로드, 프리페치 상태.
- 링크 hover, 컨텍스트 메뉴 프리로드 의도, foreground 탭 활동, 최근 활성 시간, 스케줄링에 사용되는 미디어/활동 상태.
- 북마크 기반 프리로드 기능이 활성화된 경우 북마크 제목과 URL.
- 확장 프로그램 설정, 프리로드 제한, 안전 설정, 로컬 기록 통계, AI 제공자 설정, 진단이 활성화된 경우 진단 로그.

이 로컬 데이터는 예측, 순위 계산, 프리로드 스케줄링, 안전 필터링, 로컬 기록 삭제, 진단, 선택적 AI 보조 점수 계산 기능을 제공하는 데만 사용됩니다.

## 기기 밖으로 전송되는 데이터

Smart Preload는 브라우징 기록이나 프리로드 기록을 개발자에게 전송하지 않습니다.

다음 경우에만 데이터가 기기 밖으로 나갈 수 있습니다.

- 외부 AI 제공자를 활성화하고 API key 또는 엔드포인트를 입력한 경우, 확장 프로그램은 키워드 또는 관련성 점수 계산에 필요한 선택된 페이지/링크 컨텍스트를 해당 제공자에게 보낼 수 있습니다. 해당 요청에는 그 제공자의 개인정보 처리방침이 적용됩니다.
- LM Studio 같은 로컬 AI 엔드포인트를 사용하는 경우, 요청은 사용자가 설정한 엔드포인트로 전송됩니다.
- Windows 동반 앱 업데이트를 확인하거나 다운로드하는 경우, 확장 프로그램 또는 Windows 동반 앱이 GitHub 릴리스 페이지나 GitHub 호스팅 파일에 접속할 수 있습니다.
- 실제로 페이지를 방문하거나 프리로드 기능이 브라우저에서 페이지를 로드하는 경우, 일반적인 브라우저 네트워크 통신이 발생합니다. 대상 웹사이트는 일반 페이지 로드와 마찬가지로 요청, 쿠키, 세션 정보를 받을 수 있습니다.

## 선택 사항인 Windows 동반 앱

선택 사항인 Windows 동반 앱은 실제 프리로드 창을 숨긴 상태로 유지하고 로컬 시스템 통합을 지원합니다. 같은 기기에서 Chrome/Edge native messaging을 통해 확장 프로그램과 통신합니다. 프리로드 동작을 관리하기 위해 로컬 브라우저 창 메타데이터와 시스템 성능/활동 정보를 처리할 수 있습니다. 브라우징 기록을 개발자에게 보내지 않습니다.

## 저장 및 삭제

Smart Preload는 브라우저 확장 프로그램 로컬 저장소와 확장 프로그램 또는 동반 앱이 사용하는 관련 로컬 파일에 데이터를 저장합니다. 확장 프로그램 설정 페이지에서 선택한 기간의 로컬 기록을 삭제할 수 있습니다. 확장 프로그램을 제거하거나 브라우저 프로필의 확장 프로그램 데이터를 지워 저장된 데이터를 제거할 수도 있습니다.

## 권한

Smart Preload는 기능 제공에 필요한 권한만 요청합니다. 여기에는 탭 및 탐색 이벤트 처리, 로컬 저장소, 북마크 기반 프리로드 기능, 선택 사항인 동반 앱과의 native messaging, 예약된 유지 관리 확인이 포함됩니다.

## 판매 또는 광고 사용 없음

개발자는 추적, 프로파일링 또는 광고를 위해 사용자 데이터를 판매, 대여하거나 광고주, 분석 데이터 브로커 또는 기타 제3자와 공유하지 않습니다.

## Chrome Web Store Limited Use

Smart Preload가 Chrome 확장 API에서 받은 정보의 사용은 로컬 예측과 프리로드를 통해 체감 페이지 로딩 대기 시간을 줄인다는 단일 목적을 제공하고 개선하는 데로 제한됩니다. Smart Preload의 이러한 정보 사용은 Limited Use 요구사항을 포함한 Chrome Web Store User Data Policy를 준수합니다.

## 아동 개인정보

Smart Preload는 아동을 대상으로 하지 않으며, 아동의 개인정보를 고의로 수집하지 않습니다.

## 변경 사항

Smart Preload의 데이터 처리 방식이 변경되면 이 정책이 업데이트될 수 있습니다. 최신 버전은 이 저장소에 게시됩니다.
